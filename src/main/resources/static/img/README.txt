LOGO greenField
================

Dépose ici le fichier du logo envoyé, nommé exactement :

    logo.png

(format PNG à fond transparent recommandé — aucun fond n'est ajouté par le code.)

Chemin complet attendu :
    src/main/resources/static/img/logo.png

Il est déjà câblé dans :
  - le header du front office  (front/fragments/layout.html)
  - la sidebar du back office  (back/fragments/layout.html)

Tant que le fichier est absent, l'ancien icône (feuille) s'affiche
automatiquement en repli. Après avoir déposé le fichier, redémarre
l'application (ou relance `mvn spring-boot:run`) pour qu'il soit servi.
